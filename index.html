🚀 Setup dalam 5 Menit

Defense VO1 terdiri dari Frontend (dashboard) + Backend API (server.js). Setelah API disimpan, dashboard otomatis terhubung dan mulai memantau.

1
Install dependensi
npm install
2
Buat .env
cp .env.example .env # Edit API_KEY di file .env
3
Jalankan server
npm start # ✅ HTTP : http://localhost:3000 # ✅ WS : ws://localhost:3001
4
Sambungkan dashboard → klik ⚙ API → isi URL & Key → Simpan
Dashboard akan otomatis connect dan start monitoring. Config tersimpan di localStorage.
5
Test
curl -X POST http://localhost:3000/api/sms/test

📡 Endpoint API

Semua endpoint JSON. Auth: Authorization: Bearer API_KEY

POST/api/sms/incoming🔒 Auth
Terima SMS dari Android. Server analisis → broadcast WS.
curl -X POST http://localhost:3000/api/sms/incoming \ -H "Authorization: Bearer defense-vo1-secret-key" \ -H "Content-Type: application/json" \ -d '{"sender":"+6281234","body":"Verifikasi akun: http://wa-hack.xyz"}'
POST/api/sms/test✓ Public
Kirim pesan uji acak tanpa auth — untuk test koneksi dashboard.
GET/api/sms/latest?since=ISO&limit=50&risk=all🔒 Auth
Polling — ambil pesan baru sejak timestamp tertentu.
GET/api/status✓ Public
Health check. Kembalikan uptime, jumlah pesan, klien WS.
DEL/api/sms/clear🔒 Auth
Hapus semua log dan reset statistik.
